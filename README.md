# 魔法陣ジェネレーター

ブラウザで動作するインタラクティブな魔法陣作成ツール。

**デモ:** https://tsubokulab.github.io/MagicCircleGenerator/

作者: [@kohack_v](https://x.com/kohack_v)

---

## 機能

- **3種類のレイヤー**: 円・図形（丸/弧/四角/三角/ひし形/星）・線（多角形/接続線/星形/放射状）
- **レイヤー管理**: 表示/非表示・並び替え（ドラッグ＆ドロップ）・複製・名前変更・編集
- **Undo/Redo**: Ctrl+Z / Ctrl+Y で操作履歴を管理（最大30件）
- **リアルタイムプレビュー**: スライダー操作中にゴーストガイドを表示
- **ランダム生成**: ランダムな魔法陣を自動生成
- **書き出し**: PNG / SVG / ノーマルマップ / JSON (64px〜8192px)
- **JSON インポート/エクスポート**: レイヤーデータを保存・読み込み

## 操作方法

### レイヤーの追加

1. 左パネルの「円」「図形」「線」タブを選択
2. パラメータを設定
3. 「追加」ボタンをクリック

### レイヤーの編集

右パネルのレイヤーリストで：

| 操作 | 方法 |
|---|---|
| 選択 | レイヤー行をクリック |
| 編集 | ✏️ ボタンをクリック → パラメータを変更 → 「更新」 |
| 複製 | コピーボタンをクリック |
| 名前変更 | レイヤー名をダブルクリック |
| 並び替え | ドラッグ＆ドロップ、または ↑↓ ボタン |
| 表示切替 | 👁️ アイコンをクリック |

### キーボードショートカット

| キー | 操作 |
|---|---|
| `Ctrl+Z` | 元に戻す |
| `Ctrl+Y` / `Ctrl+Shift+Z` | やり直す |

### 書き出し

1. 右パネル「ダウンロード」でフォーマットとサイズを選択
2. 「ダウンロード」ボタンをクリック

## 開発

```bash
# 依存関係インストール
npm install

# GitHub Pages へデプロイ
npm run deploy
```

ローカルで確認する場合は `index.html` をブラウザで直接開くか、ローカルサーバーを起動してください。
